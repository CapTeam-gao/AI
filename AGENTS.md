# 절대 하지 말아야 할 것들
- 내 허락 없이 파일을 삭제하지 말 것.
- 모르면 추측해서 진행하지 말고 먼저 물어볼 것.

# 작업 방식
- 한 번에 하나의 기능만 만들 것.
- 수정하기 전에 어떤 파일을 바꿀지 먼저 말할 것.
- 작업이 끝나면 바꾼 파일과 확인한 내용을 짧게 설명할 것

# progress.txt 사용 규칙
- 새 작업을 시작하면 `progress.txt`에 해야 할 일을 최대 2줄로 적을 것.
- 작업을 완료하면 완료한 할 일 줄은 `progress.txt`에서 지울 것.
- 작업을 완료한 할일만 지울 것
- 아직 남은 작업이 있으면 다음에 해야 할 일만 1~2줄로 남길 것
